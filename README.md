# Iris-Species-predictor
Assignment 2 
This project is based on machine learning where I trained a model to predict the species of an Iris flower.

I used the Iris dataset to train a Multi-Layer Perceptron (MLP) model in Python. After training, I converted the model into TensorFlow Lite format so it can run inside a mobile application.

The app takes four inputs (sepal length, sepal width, petal length, petal width) and shows the predicted flower species (Setosa, Versicolor, or Virginica).

For running the mobile app and generating a QR code, I also used Snack Expo, which made it easy to test the application on my mobile device by scanning the QR code.

This project helped me understand how machine learning models can be trained and deployed in real-world mobile applications.
